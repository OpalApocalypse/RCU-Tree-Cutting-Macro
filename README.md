# RCU Tree Cutting Macro

Automates cutting trees in RCU using AutoHotkey — gathers resources and farms XP to raise mastery level incredibly well.

---

> [!WARNING] This Macro is on a hiatus/hold status for now - this version works, but may have issues that won't be fixed.
---

## Hotkeys

| Hotkey | Action | Notes |
|--------|--------|-------|
| <kbd>F1</kbd> | Start macro | Starts the macro and opens the menu |
| <kbd>F2</kbd> | Pause / Resume | Toggles pause state |
| <kbd>F3</kbd> | Zone Selection menu | Opens zone selection menu. Does not load after selection, must press F1 again. |
| <kbd>F4</kbd> | Stop macro | Terminates the macro script |

---

## How to Use Macro

- Load into a fresh game session and do **not** move your character or camera.
- Press F1 to start the macro immediately for farming Spawn.
- Other zones were in the works but are not ready for use yet. The F3 menu is effectively not useful in this version.
